## What This Project Is

AI Project Template is the permanent engineering foundation (Oster Engineering System) for all future software projects. It is a GitHub Template Repository — infrastructure only, no application code. Child projects are created via **Use this template** on GitHub.

## Current Focus

Building the `.project/` memory system and bootstrap templates so child projects can inherit project identity, build rules, decision logs, history, and checklists.
